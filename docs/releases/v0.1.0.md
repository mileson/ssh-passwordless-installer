# v0.1.0

Initial standalone public release of the SSH passwordless setup helper.

## Highlights

- macOS double-click installer script
- Windows `.bat` plus `.ps1` installer flow
- standalone macOS `.app` packaging script
- release bundle builder for both macOS and Windows
- bilingual README set, preview asset, and community files

## Installation

### Run from source

```bash
chmod +x ./scripts/macos/setup-passwordless-ssh.command
./scripts/macos/setup-passwordless-ssh.command
```

Windows users can run `scripts\windows\setup-passwordless-ssh.bat` directly.

### Release assets

- `SSH-Passwordless-Setup-macOS.zip`
- `SSH-Passwordless-Setup-Windows-Download-Then-Double-Click.zip`

## Verification

- confirm a new SSH key pair is created or an existing alias-specific key is reused
- confirm the public key is installed on the target server
- confirm the managed `~/.ssh/config` block is written correctly
- confirm passwordless login works both by direct key and by alias

## How To Use

1. enter server IP or hostname
2. enter SSH username
3. enter a local alias
4. type the server password directly into SSH
5. use the new alias for passwordless login
